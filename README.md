# Ionic 1 Examples

This is a collection of examples for Ionic 1.

## Installation

Ionic 1 doesn't work well with Node 5.x, so use nvm to install the latest version of node 4.x.

```
nvm install 4
npm install -g cordova ionic
npm install -g ios-sim android-sim
```
